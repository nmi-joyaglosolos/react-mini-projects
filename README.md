# React Mini Projects

A collection of small React exercises and mini projects built to practice React fundamentals like components, hooks, props, and state management.

---

- [**Basic Advice App**](https://github.com/nmi-joyaglosolos/basic-advice) – Fetches and displays random advice from a public API using React hooks.

- [**Steps App**](https://github.com/nmi-joyaglosolos/steps) – A simple step-by-step interface (1–2–3) with buttons to move forward or backward.

- [**Pizza Menu**](https://github.com/nmi-joyaglosolos/pizza-menu) – Displays a list of pizzas with names, prices, and stock info; items are clickable for interaction.

- [**Flashcards App**](https://github.com/nmi-joyaglosolos/flashcards) – Flashcard functionality with questions and answers; only one card can be flipped at a time.

- [**Date Skip Counter App**](https://github.com/nmi-joyaglosolos/date-counter) – Lets users move forward or backward in time by changing dates with buttons.

- [**Travel Essentials Tracker**](https://github.com/nmi-joyaglosolos/travel-list) – Tracks travel items to pack, mark as packed, sort lists, and view remaining items.

---

Each project focuses on a specific React concept while keeping the implementation simple and beginner-friendly.
